# 🏭 Warehouse Management System (WMS)

A full-featured Warehouse Management System built with **ASP.NET Core MVC** using **N-Tier Architecture**.

## 🛠️ Tech Stack
- **Backend:** ASP.NET Core MVC (.NET 10)
- **Architecture:** N-Tier (Models, DAL, BLL, Web)
- **Database:** SQL Server + Entity Framework Core
- **Authentication:** BCrypt Password Hashing + Session
- **Email:** Gmail SMTP
- **Reports:** EPPlus (Excel)
- **UI:** Bootstrap 5

## ✅ Features
- 🔐 Authentication & Authorization (Login/Register)
- 📦 Product Management (Add/View/Delete)
- ⚖️ Warehouse Capacity Validation (Max 1000 items / 3 Tons)
- 💰 Automatic Storage Fee Calculation
- 📧 Email Notifications (Entry/Exit/Shipping)
- 📊 Excel Reports (Products/Fees/Logs)
- 📋 Activity Logs
